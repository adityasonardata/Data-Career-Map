# 🚀 Data Roles Career Map

An interactive web app to explore different careers in the data field — including roles, salaries, skills, and a quick quiz to find your best fit.

---

## 🔗 Live Demo

👉 https://github.com/adityasonardata/Data-Career-Map/blob/main/index.html

---

## 📊 What this project includes

* 📌 11 Data roles (Analyst → AI Engineer)
* 💰 Salary ranges (India market - 2026)
* 🛠 Skills & tools required for each role
* 📅 Day-to-day work breakdown
* 🧭 Career tiers (Core, Related, Emerging)
* 🧠 Interactive quiz to find your ideal role

---
